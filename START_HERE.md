## 🚀 Deploy Now

Choose your preferred platform:

### Docker (Fastest)
```bash
docker-compose up -d
```

### Vercel (Easiest)
```bash
npm install -g vercel
vercel deploy
```

### Railway
```bash
npm install -g @railway/cli
railway up
```

### Heroku
```bash
git push heroku main
```

```env
NEXT_PUBLIC_AGENT_NAME=Trainco AI
```
